Known issues / gotchas

BackendMust run separately: cd api && go run main.go
currentValueSet by backend at write time — not recalculated on read. Needs price refresh logic.
Stock symbolBackend appends .NS — pass bare symbol e.g. RELIANCE not RELIANCE.NS
MF lookupScheme code from mfapi.in — not AMFI code directly
Gold fallbackReturns ₹7200/g if GOLD_API_KEY missing or API fails
Auth tokenportfolio.ts reads from idToken store (auth.ts), not localStorage
Firestore pathusers/{uid}/assets/{assetId}
Asset IDGenerated as timestamp string: time.Now().Format("20060102150405")
